# SuperDesktop
Graphical launcher for links and folders, using a complete virtual desktop manager. It has diferent desktop where you can put documents, images, photos, videos,... And links to websites. It's has folders to organized a lot of items. It has widgets like images (that you can adapt to any size and form), videos playing in loop and without any sound, notes, and a transparent separator of items.

You can customize every items icons, any size of the widgets, number of desktops (and columns of desktops), wallpaper of every desktop, wallpaper mode (scaled or maximized), time to change of wallpaper (if you choose a folder path instead a single image wallpaper path),...
